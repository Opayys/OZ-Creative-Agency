PROJECT 01 / 04
⬡ Solana Ecosystem

Axeon Management Protocol

"Hybrid gatekeeping infrastructure untuk komunitas Telegram premium — bayar lewat QRIS atau kripto, akses dikontrol otomatis."

Admin grup Telegram premium biasanya mengecek bukti transfer satu per satu, dan sering kelambatan saat harus keluarkan member yang expired. Axeon mengganti proses itu dengan smart contract: setiap pembayaran masuk on-chain, divalidasi otomatis, dan bot melakukan scan berkala untuk auto-remove member yang masa langganannya habis.

  • Unified Hybrid Payment — QRIS/Bank (Web2) dan Kripto (Web3) dalam satu alur.
  • Automated Validation — verifikasi pembayaran real-time, tanpa cek manual.
  • Hourly Guard Bot — scan tiap 1–60 menit, auto-remove member expired.
  • ZK Privacy — detail transaksi dilindungi Zero-Knowledge Proof Solana.
  • Solana PDA — biaya transaksi sangat rendah dan cepat.

PROJECT 02 / 04
🎮 Solana · GameFi

Stack Block Arcade

"Skill-to-earn arcade — gameplay berjalan di browser, hadiah dikunci dan diverifikasi di blockchain."

Game Web3 biasa terjebak di antara dua masalah: kalau semua transaksi on-chain, game jadi lag dan gas fee mahal; kalau semua off-chain, pemain bisa cheat. Stack Block Arcade memisahkan keduanya — pemain beli tiket dan klaim reward lewat smart contract, tapi gameplay-nya sendiri berjalan mulus di browser tanpa transaksi sama sekali.

  • Hybrid Web2.5 — gameplay 60 FPS di browser, transaksi keuangan on-chain.
  • Anti-Cheat Engine — server backend mensimulasikan ulang log koordinat & timestamp.
  • Cryptographic Payout — reward dikunci signature digital, diverifikasi smart contract.
  • Leaderboard Prize Pool — pool mingguan dikunci on-chain, distribusi otomatis ke top player.
  • Tiered P2E — makin tinggi skor, makin besar return dari harga tiket.

⟠ Ethereum Ecosystem

VeritasVote

"Governance terdesentralisasi — tidak ada admin yang bisa mengubah atau menghapus suara siapa pun."

Sistem voting konvensional rentan terhadap dua masalah: kepercayaan terpusat pada satu pihak, dan manipulasi lewat flash loan (pinjam token sesaat hanya untuk vote). VeritasVote menyelesaikan keduanya dengan snapshot voting power di blok pembuatan proposal, dan eksekusi otomatis begitu proposal disetujui mayoritas — lengkap dengan timelock pengaman.

  • Snapshot Anti-Flashloan — hak suara dikunci pada blok saat proposal dibuat.
  • Fuzz Testing — diuji ribuan input acak untuk cegah integer overflow & double voting.
  • Timelock Execution — proposal yang lolos dieksekusi otomatis dengan jeda pengaman.
  • For / Against / Abstain — tiga opsi vote dengan kalkulasi kuorum otomatis.
  • Wagmi + Viem — dashboard reaktif ringan, hooks blockchain-native.
